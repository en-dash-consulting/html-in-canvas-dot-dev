Showing layout A — ready to morph

Select a transition mode, then click Morph to blend between layouts

1. Two distinct HTML layouts live inside a <canvas layoutsubtree> — each with unique structure and styling

2. drawElementImage() renders each layout into the canvas bitmap separately

3. getImageData() captures both as raw pixel buffers for per-pixel blending

4. Custom transition algorithms blend the pixels frame-by-frame: crossfade, dissolve, wave wipe, or pixel sort

These transitions work across entirely different DOM structures and text content — something CSS transitions simply cannot do.
